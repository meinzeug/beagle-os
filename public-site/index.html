Everything you need for managed virtual desktops

One platform. Host install, fleet control, endpoint runtime, and release automation.

Dedicated Endpoint OS

Purpose-built Debian-based thin client OS. Boots directly into Moonlight — no desktop, no distractions, no attack surface.

Beagle Control Plane

Manage virtualization, endpoints, updates, policies, and downloads from one Beagle-native control plane.

GPU-Accelerated Streaming

Moonlight + Sunshine deliver sub-5ms latency with H.264, HEVC, or AV1. Supports up to 4K@120fps.

Fleet Management

Control plane with health checks, endpoint reporting, bulk actions, and policy management for your entire fleet.

Secure by Design

Per-VM SSH tunnel keys, scoped endpoint tokens, enrollment-based onboarding. No open ports, no shared secrets.

USB Passthrough

Securely tunnel local USB devices into your VM through encrypted SSH reverse tunnels. Webcams, printers, security keys.

Up and running in minutes

From bare metal or lab VM to streamed desktop in five steps.

1

Install the host

Boot the server installer ISO or run the host installer to provision the Beagle host and control plane.

2

Prepare your VM

Configure Sunshine on your Windows or Linux VM. Beagle auto-detects GPU capabilities and optimizes settings.

3

Download the Installer

Generate a VM-specific USB live or installer script. The endpoint profile is embedded automatically.

4

Boot the Endpoint

Flash to USB, boot your thin client. Beagle OS starts, pairs with Sunshine, and launches the stream.

5

Manage from Beagle Control Plane

Monitor health, push policies, rebuild media, and manage your fleet from the Beagle WebUI.
