FAQ

The most common questions about licensing and hosted access.

Does the hosted path replace self-hosting?

No. Beagle remains Beagle-native. The hosted path is the commercial service layer around that deployment model.

Do endpoints still boot directly into Moonlight?

Yes. The endpoint OS path is unchanged. The thin client still boots directly into Moonlight against the assigned VM.

Can I use Beagle OS privately for free?

Yes. Private and non-commercial use stays free. Commercial use requires a license request first.

Where do I check the terms?

Use the LICENSE file in the GitHub repository as the source of truth, and contact us for commercial approval if needed.
