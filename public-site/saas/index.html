What SaaS adds

Commercial licensing without changing the technical deployment model

You still run the Beagle-native control plane and still connect endpoints with Moonlight to Sunshine. The SaaS governs the commercial account and service layer.

Self-hosting

Private and non-commercial use stays free

Use the repository, host setup scripts, and self-hosted artifact flow if you are not operating commercially.

FAQ

The most common questions about the hosted commercial path.

Does the SaaS replace the self-hosted control plane?

No. Beagle remains Beagle-native. The SaaS is the commercial service layer around that deployment model.

Do endpoints still boot directly into Moonlight?

Yes. The endpoint OS path is unchanged. The thin client still boots directly into Moonlight against the assigned VM.

Can I start with self-hosting and move later?

Yes. The technical product path stays the same. The difference is the license and service layer for commercial operation.

Where do I log in?

Use the hosted account flow at beagle.meinzeug.cloud for login, registration, and dashboard access.
